# AVL-дерево для системы управления меню

Курсовая работа по дисциплине «Алгоритмы и структуры данных»

---

## О проекте

Программа для управления меню столовой, которая хранит названия блюд и цены.

**Основные возможности:**
- быстрый поиск цены по названию
- вывод всего меню в алфавитном порядке
- добавление и удаление блюд

---

## Почему это интересно с точки зрения разработки

### 1. Выбор структуры данных под конкретную задачу

В работе сравнила 10 разных способов хранения данных. Критерии: 85% операций — поиск, 10% — вывод всего меню, 5% — добавление/удаление.

| Структура данных | Поиск | Добавление/удаление | Сортировка | Гарантии |
|------------------|-------|---------------------|------------|----------|
| Хеш-таблица | O(1) | O(1) | не поддерживает | нет |
| Красно-черное дерево | O(log n) | O(log n) | поддерживает | есть |
| **AVL-дерево** | **O(log n) быстрее** | O(log n) | поддерживает | **строгий баланс** |

AVL-дерево обеспечивает самый строгий баланс (высота ≤ 1.44·log₂n), поэтому поиск работает быстрее, чем в других сбалансированных деревьях.

### 2. Архитектура программы

Код разделён на логические части:
- **MenuManager** — внешний интерфейс для работы с меню
- **AVL-дерево** — основные операции (вставка, удаление, поиск)
- **Стек и узлы** — внутренние механизмы для обхода дерева

### 3. Детали реализации

- **Стек написан вручную** — без использования встроенных списков Python, на связном списке
- **Обход дерева без рекурсии** — итератор выдаёт элементы по одному, не собирая их в память заранее
- **Балансировка после каждой операции** — дерево всегда остаётся сбалансированным

### 4. Тестирование

20 тестов проверяют:
- добавление, поиск, обновление, удаление
- работу с пустым деревом и граничные случаи
- регистронезависимость (поиск "борщ" и "БОРЩ" работает одинаково)
- балансировку после множества операций

### 5. Исследование производительности

Для массивов от 10 до 1000 элементов замерила время операций. Результаты совпадают с теорией: поиск, вставка, удаление работают за O(log n), обход всего дерева — за O(n).

### 6. Визуализация

С помощью Graphviz можно увидеть структуру дерева в любой момент — наглядно видно, что оно сбалансировано и отсортировано по алфавиту.

## Пример использования

```python
from src.cw import MenuManager

menu = MenuManager()
menu.add("Борщ", 150)
menu.add("Солянка", 180)

price = menu.get_price("Борщ")        # 150

menu.remove("Солянка")

for name, price in menu:
    print(f"{name}: {price} руб.")
```
